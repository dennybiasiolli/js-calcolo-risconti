# calcolo-risconti

Una semplice utility per calcolare ratei o risconti suddivisi per anno.
Implementata in HTML, CSS e JavaScript vanilla (nessun framework).

***

## Quick Start

Apri nel browser:

`file:///path/to/js-calcolo-risconti/CalcoloRisconti.html`

oppure servi la cartella con un server locale statico.
